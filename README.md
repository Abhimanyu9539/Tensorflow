# TensorFlow Learning Repository

Welcome to my TensorFlow Learning Repository! This collection showcases my journey through various TensorFlow projects, encompassing machine learning, deep learning, and computer vision applications.

## Repository Structure

The repository is organized into several folders, each representing a distinct project or concept explored during my learning process:

1. **CNN-Image-Classification-Real-Time**: Implements real-time face recognition using Convolutional Neural Networks (CNNs).

2. **Covid19_detection_using_X_ray_images**: Develops a model to detect COVID-19 from X-ray images.

3. **Mask-Recognition**: Creates a real-time mask recognition system.

4. **Recommender_Engines**: Explores different recommender engine algorithms.

5. **Facial-Expression-Recognition-with-Keras-and-deployment-using-flask**: Builds a facial expression recognition system with Keras and deploys it using Flask.

6. **NLP_Resume_Parser**: Constructs a resume parser in Python using SpaCy.

## Getting Started

To explore any project:

1. **Clone the Repository**:

   ```bash
   git clone https://github.com/Abhimanyu9539/Tensorflow.git
   ```

2. **Navigate to the Project Directory**:

   ```bash
   cd Tensorflow/<project-folder>
   ```

3. **Follow the Instructions**:

   Each project folder contains a README file with detailed instructions on setup, dependencies, and usage.

## Prerequisites

- **Python 3.x**: Ensure Python is installed on your system.

- **TensorFlow**: Install TensorFlow by following the [official installation guide](https://www.tensorflow.org/install).

- **Additional Libraries**: Refer to each project's README for specific dependencies.
